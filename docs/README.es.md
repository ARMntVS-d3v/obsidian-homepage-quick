# HomePage Quick

**HomePage Quick** es un plugin simple y rápido para Obsidian que permite establecer una página de inicio que se abre automáticamente al iniciar la aplicación. Funciona perfectamente en todos los dispositivos, incluidos los móviles.

## 🌍 Idiomas Disponibles
- [English](../README.md)
- [Русский](README.ru.md)
- [简体中文](README.zh.md)

## 🚀 Características
- Selecciona cualquier archivo en Obsidian como tu página de inicio.
- Se abre automáticamente al iniciar la aplicación.
- Optimizado para un rendimiento rápido en todos los dispositivos.

## 🔧 Instalación
1. Abre **Configuración** en Obsidian.
2. Ve a la sección **Plugins de la comunidad**.
3. Busca **HomePage Quick** e instálalo.
4. Activa el plugin después de la instalación.

## ⚙️ Configuración
El plugin tiene una única configuración: elige el archivo que se usará como página de inicio.
