# HomePage Quick

**HomePage Quick** 是一个简单快捷的 Obsidian 插件，允许您设置一个主页，在启动应用程序时自动打开。它可以在所有设备上无缝运行，包括移动设备。

## 🌍 可用语言
- [English](../README.md)
- [Русский](README.ru.md)
- [Español](README.es.md)

## 🚀 功能特点
- 选择 Obsidian 中的任何文件作为主页。
- 启动应用程序时自动打开所选文件。
- 针对所有设备进行了速度优化。

## 🔧 安装
1. 打开 Obsidian 的 **设置**。
2. 转到 **社区插件** 部分。
3. 搜索 **HomePage Quick** 并安装它。
4. 安装完成后，启用该插件。

## ⚙️ 设置
插件只有一个设置：选择要用作主页的文件。
